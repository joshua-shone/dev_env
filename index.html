Setup window interactions

With most desktop window managers, resizing windows requires hitting a tiny margin around the window border to start a drag. Closing or minimizing means you have to bullseye a tiny 10x10 pixel button.

Why not use the whole window as a cursor target?

Out of the box, KDE allows for resizing windows using an alt-rightclick drag from anywhere over the window surface. After changing a setting, you can also close windows by middle-clicking anywhere on a window's titlebar.

Now, rather than window resizing being an occasional concerted effort, it can be done continuously to keep windows logically arranged and grouped as you work.

Clean up Dolphin

There's a huge amount of clutter that can be cleared up in KDE's file-manager, Dolphin. Once you know the shortcut keys, the menus, toolbars and panels are largely superfluous.

The result is a compact frame - of which many can be layed out on a desktop in a logical fashion, rather than having a single window take most of the screen.

Go to View > Panels and uncheck everything. Also uncheck 'Show Toolbar' in the settings menu.

Hit Ctrl + M to hide the menu-bar too.

Setup application shortcuts

Go to System Settings > Shortcuts > Custom Shortcuts.

Add a new entry by clicking Edit > New > Global Shortcut > Command/URL. Set the shortcut keys in Trigger and type in the application name in Action.

I reckon the Unix philosopy ought to extend to graphical desktop land, namely 'Do one thing and do it well'.

Rather than use an IDE, why not use a core set of essential utilities. With a file-manager, terminal, text-editor and web-browser, pretty much anything can get done.

The desktop can be an IDE; just open apps like you would tabs.

Virtual desktops

There's a few tweaks you can do that makes virtual desktops feel extra fluid and natural, as if they really were extensions of your monitor(s).

Turning on the 'Slide' desktop switch animation enables some spatial continuity and avoid some "pause-and-reorient-yourself" moments.

You can also enable desktop switching when a window is dragged to the edge of the screen. It means you can grab a window a push it down until it rolls onto the next desktop, like you really did have a screen the size of a highway billboard.
